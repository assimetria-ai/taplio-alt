## New Features Added

### 1. **InvitationBadge Component** 🔴
Shows a red notification badge with the count of pending invitations.

**Usage:**
```jsx
<nav>
  <Link to="/app/teams">
    Teams <InvitationBadge />
  </Link>
</nav>
```

**Result:** Users see "3" badge next to Teams link when they have pending invitations.

---

### 2. **useTeamInvitations Hook** 🪝
Reusable React hook for managing invitation state.

**Features:**
- Auto-fetches pending invitations on mount
- Provides invitation count for badges
- Includes `refresh()` and `acceptInvitation()` functions
- Handles loading and error states

**Usage:**
```jsx
const { 
  invitations,       // Array
  invitationCount,   // Number
  loading,           // Boolean
  refresh,           // Function
  acceptInvitation   // Async function
} = useTeamInvitations()
```

## How It Works

### User Flow

1. **User B receives email invitation to join Team A**
2. **User B logs in and navigates to any page**
   - Sees red "1" badge on Teams navigation link
3. **User B clicks Teams → lands on `/app/teams`**
   - Pending invitations section displays at top
   - Shows: Team name, inviter, role, expiration date
4. **User B clicks "Accept"**
   - Confirmation dialog: "Accept invitation to join 'Team A'?"
   - Button shows loading state during acceptance
5. **Invitation accepted**
   - User B is now a team member
   - Invitation disappears from pending list
   - Badge updates automatically
