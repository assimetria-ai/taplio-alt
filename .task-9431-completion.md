## Fix Applied

### Issue

The `EmailLogRepo` and `FileUploadRepo` classes existed but were not exported from the central repository index file, making them inaccessible for standard imports.

### Solution

Added missing exports to `server/src/db/repos/@custom/index.js`:

```javascript
const EmailLogRepo = require('./EmailLogRepo')
const FileUploadRepo = require('./FileUploadRepo')

module.exports = { 
  // ... existing repos
  EmailLogRepo,
  FileUploadRepo
}
```

### Commit

