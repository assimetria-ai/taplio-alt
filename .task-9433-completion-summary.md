- Appears as centered dialog on desktop
- Includes drag handle for mobile UX
- Portal-based rendering
- Keyboard navigation (Escape to close)
- Body scroll prevention
- Touch-friendly actions with proper sizing

```jsx

</BottomSheet>
```
