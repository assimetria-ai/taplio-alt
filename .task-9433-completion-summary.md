## Migration Path

### For Existing Components
1. Update forms to use `MobileForm` wrapper:
   ```jsx
   // Before
   <form>
     <label>Name</label>
     <input />
   </form>

   // After
   <MobileForm>
     <MobileForm.Field label="Name">
       <Input />
     </MobileForm.Field>
   </MobileForm>
   ```

2. Replace tables with `MobileTable`:
   ```jsx
   // Before
   <DataTable columns={cols} data={data} />

   // After  
   <MobileTable columns={cols} data={data} />
   ```

3. Use `MobileModal` for dialogs:
   ```jsx
   // Before
   <Dialog>...</Dialog>

   // After
   <MobileModal>...</MobileModal>
   ```
