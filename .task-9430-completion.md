## Features & Benefits

### For Developers:

✅ **Zero Boilerplate Option**
```javascript
module.exports = createCrudRouter({
  repo: ProductRepo,
  config: { basePath: '/api/products' }
})
```
This creates 5 endpoints instantly (LIST, GET, CREATE, UPDATE, DELETE).

✅ **Automatic Pagination**
```javascript
router.get('/api/products', pagination(), handler)
```
Supports both offset and page-based pagination.

✅ **Built-in Search**
```javascript
const search = parseSearchQuery(req.query, {
  defaultFields: ['name', 'description']
})
```

✅ **Smart Filtering**
```javascript
const queryConfig = parseQueryParams(req, {
  searchFields: ['name', 'description'],
  sortableFields: ['name', 'price', 'created_at'],
  filterFields: ['category', 'status'],
  booleanFields: ['is_active'],
})
```

✅ **Query Builder Protection**
All query builders use parameterized queries (SQL injection safe).

### API Capabilities Out of the Box:

| Feature | Example |
|---------|---------|
| Search | `GET /api/products?q=laptop` |
| Filtering | `GET /api/products?category=electronics&in_stock=true` |
| Sorting | `GET /api/products?sort=price&order=asc` |
| Pagination | `GET /api/products?page=2&limit=20` |
| Field Selection | `GET /api/products?fields=name,description` |
| Combined | `GET /api/products?q=gaming&category=electronics&sort=price&order=asc&page=1&limit=20` |
