## Testing

### Manual Verification

The todos-example.js can be tested by:

1. Adding to routes:
```javascript
// In server/src/routes/@custom/index.js
router.use(require('../../api/@custom/todos-example'))
```

2. Starting server and testing endpoints:
```bash
# List todos (with pagination)
curl http://localhost:3000/api/todos?limit=10&page=1

# Create todo
curl -X POST http://localhost:3000/api/todos \
  -H "Content-Type: application/json" \
  -d '{"title":"Test todo","priority":"high"}'

# Get by ID
curl http://localhost:3000/api/todos/1

# Update
curl -X PATCH http://localhost:3000/api/todos/1 \
  -H "Content-Type: application/json" \
  -d '{"completed":true}'

# Delete
curl -X DELETE http://localhost:3000/api/todos/1
```
