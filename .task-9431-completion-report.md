## API Endpoints Verified

### Email Logs API
```
GET    /api/email-logs               — List all email logs (admin)
GET    /api/email-logs/:id           — Get single log
GET    /api/email-logs/stats         — Email statistics
GET    /api/email-logs/volume        — Daily volume chart
GET    /api/email-logs/templates     — Template breakdown
GET    /api/email-logs/preview       — List preview templates
GET    /api/email-logs/preview/:name — Render template preview
POST   /api/email-logs               — Ingest email event (shared secret)
PATCH  /api/email-logs/:id/status    — Update email status
```

### Storage API
```
POST   /api/storage/presign          — Get presigned upload URL
PATCH  /api/storage/:id/confirm      — Confirm upload
GET    /api/storage/:id              — Get file record
DELETE /api/storage/:id              — Soft delete file
POST   /api/storage/:id/restore      — Restore deleted file
```

### Audit Logs API

GET    /api/audit-logs/user/:userId         — User activity
POST   /api/audit-logs                      — Create audit log (admin)
