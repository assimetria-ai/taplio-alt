**Features:**
- **Multi-provider storage:**
  - AWS S3
  - Cloudflare R2
  - Local filesystem (development)
- **Direct browser-to-cloud uploads:**
  - Presigned PUT URLs
  - Zero server bandwidth usage
  - Parallel uploads support
- **File tracking:**
  - Database records for all uploads
  - User ownership tracking
  - File metadata (size, type, original name)
  - Upload status (pending, confirmed, deleted)

  - MIME type validation
  - File size limits (100 MB default)
  - No SVG uploads (XSS prevention)
  - Allowed types: images, documents, audio, video, archives
- **File lifecycle:**
  - Soft delete (recoverable)
  - Hard delete (permanent)
  - Automatic cleanup
- **Public/Private URLs:**
  - Presigned download URLs
  - Public URL generation
  - Expiring URLs for security

**Usage Example:**
```javascript

// Create presigned upload URL

  filename: 'document.pdf',
  contentType: 'application/pdf',
  folder: 'documents'
})

// Client uploads directly to presigned URL
// PUT to url with file content

// Create presigned download URL

  key: key,
  expiresIn: 3600 // 1 hour
})

// Delete file
```

**Usage Example:**
```javascript
const logger = require('./lib/@system/Logger')

// Application logging
logger.info('User logged in', { userId: 42, ip: req.ip })
logger.error({ err }, 'Database connection failed')
logger.debug({ data }, 'Processing request')

  user_id: req.user.id,
  action: 'update_profile',
  entity_type: 'user',

  ip_address: req.ip,
  user_agent: req.get('User-Agent')
})

  action: 'update_profile',
```
