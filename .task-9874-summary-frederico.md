## API Endpoints Needed

The components expect these backend endpoints (all documented):

```
# Onboarding
PATCH /users/me/onboarding
POST /teams/invites

# Settings
GET/PATCH /users/me
GET/POST /users/me/2fa/enable
GET/DELETE /users/me/sessions/:id
GET/PATCH /users/me/notifications

# Dashboard
GET /dashboard/stats
GET /dashboard/activity
GET /dashboard/users
```

Mock data is used as placeholders (marked with `@custom` comments).
