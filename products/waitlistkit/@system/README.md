# @system - WaitlistKit System Directory

This directory contains system-level utilities and common functionality for the WaitlistKit product.

## Purpose

The `@system` directory holds:

- Shared utilities used across WaitlistKit components
- System-level configuration
- Common helpers and middleware
- QA reports and system documentation

## Current Status

**Status:** Active Product  
**Created:** 2026-03-07

## Structure

This directory will contain:

```
@system/
├── README.md          (this file)
├── QA_REPORT_*.md     (automated QA reports)
└── [utilities as needed]
```

## Development Notes

- System utilities should be framework-agnostic when possible
- Documentation should be kept up-to-date
- QA reports are auto-generated by Duarte QA system
