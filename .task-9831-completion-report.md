## 🚀 Usage Example

### Zero-Config CRUD (Auto-generate 5 endpoints)

```javascript
const { createCrudRouter } = require('./lib/@system/Helpers')

module.exports = createCrudRouter({
  repo: ProductRepo,
  config: { basePath: '/api/products' },
})
```

This single line creates:
- `GET /api/products` - List with pagination, search, filters, sorting
- `GET /api/products/:id` - Get single item
- `POST /api/products` - Create
- `PATCH /api/products/:id` - Update
- `DELETE /api/products/:id` - Delete

### Manual CRUD with Full Control

```javascript
const {
  pagination,
  sorting,
  filtering,
  handleList,
  parseQueryParams,
} = require('./lib/@system/Helpers')

router.get('/api/products',
  pagination({ defaultLimit: 20 }),
  sorting({ allowedFields: ['name', 'price', 'created_at'] }),
  filtering({ allowedFields: ['category', 'status'] }),
  async (req, res, next) => {
    const config = parseQueryParams(req, {
      searchFields: ['name', 'description'],
      sortableFields: ['name', 'price', 'created_at'],
      filterFields: ['category', 'status', 'in_stock'],
    })
    
    await handleList({
      repo: ProductRepo,
      req, res, next,
      filters: config,
    })
  }
)
```

### Query Examples

All of these work out of the box:

```bash
# Pagination
GET /api/products?page=1&limit=20
GET /api/products?offset=0&limit=50

# Search
GET /api/products?q=laptop&fields=name,description

# Filtering
GET /api/products?category=electronics&in_stock=true

# Advanced filtering with operators
GET /api/products?price[gte]=100&price[lte]=1000

# Sorting
GET /api/products?sort=price&order=asc
GET /api/products?sort=category:asc,price:desc

# Combined
GET /api/products?q=gaming&category=electronics&price[lte]=500&sort=price:asc&page=1&limit=20
```
