## Features

### Query Parameters Supported:

**Pagination:**
- `?page=2&limit=20` (page-based)
- `?offset=40&limit=20` (offset-based)

**Sorting:**
- `?sort=price&order=desc` (single field)
- `?sort=name:asc,price:desc` (multi-field)

**Filtering:**
- `?category=electronics` (simple equality)
- `?in_stock=true` (boolean)
- `?price[gte]=100&price[lte]=500` (range)
- `?tags[in]=tutorial,beginner` (array)
- `?name[like]=laptop` (pattern matching)

**Search:**
- `?q=search+term` (full-text)
- `?q=laptop&fields=name,description` (multi-field)

### Security Features:
- ✅ SQL injection protection (parameterized queries)
- ✅ Field whitelisting (prevents mass assignment)
- ✅ Input sanitization
- ✅ Type validation
- ✅ Configurable limits

### Performance:
- ✅ Database indexes recommended for sort/filter fields
