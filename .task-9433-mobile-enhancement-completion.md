## 🆕 New Components

### 1. MobileTable (`@system/Dashboard/MobileTable.jsx`)
**Purpose:** Automatically switches between table (desktop) and card (mobile) views

**Features:**
- Card view on mobile (<1024px)
- Table view on desktop (≥1024px)
- Primary field highlighting in card view
- Configurable columns with `hideOnMobile` option
- Touch-friendly row selection
- Loading skeleton state
- Custom empty state support

**Usage:**
```jsx
<MobileTable
  columns={[
    { key: 'name', label: 'Name', primary: true },
    { key: 'email', label: 'Email' },
    { key: 'status', label: 'Status', hideOnMobile: true }
  ]}
  data={users}
  onRowClick={(user) => navigate(`/users/${user.id}`)}
/>
```

### 2. MobileForm (`@system/Form/MobileForm.jsx`)
**Purpose:** Form wrapper with mobile-optimized spacing and behavior

**Features:**
- Automatic keyboard dismissal on submit
- Responsive field spacing (5/6 unit scale)
- Compound components for structure:
  - `MobileForm.Field` - Label, input, error display
  - `MobileForm.Group` - Side-by-side fields (desktop), stacked (mobile)
  - `MobileForm.Section` - Logical form sections with titles
  - `MobileForm.Actions` - Button area (stacked mobile, horizontal desktop)
- Full-width inputs on mobile
- Touch-friendly form controls

**Usage:**
```jsx
<MobileForm onSubmit={handleSubmit}>
  <MobileForm.Section title="Personal Info">
    <MobileForm.Group>
      <MobileForm.Field label="First Name" required>
        <Input {...register('firstName')} />
      </MobileForm.Field>
      <MobileForm.Field label="Last Name" required>
        <Input {...register('lastName')} />
      </MobileForm.Field>
    </MobileForm.Group>
  </MobileForm.Section>
  
  <MobileForm.Actions>
    <Button variant="outline">Cancel</Button>
    <Button type="submit">Save</Button>
  </MobileForm.Actions>
</MobileForm>
```

### 3. MobileModal (`@system/Modal/MobileModal.jsx`)
**Purpose:** Full-screen modals on mobile, centered dialogs on desktop

**Features:**
- Full-screen on mobile (100dvh)
- Centered dialog on desktop
- Size variants: sm, md, lg, xl, 2xl
- Backdrop blur overlay
- Body scroll lock when open
- ESC key to close
- Close on overlay click (optional)
- Compound components:
  - `MobileModal.Content` - Main content area
  - `MobileModal.Actions` - Button area
- Smooth animations (slide-in, fade-in, zoom)

**Usage:**
```jsx
<MobileModal
  open={isOpen}
  onClose={() => setIsOpen(false)}
  title="Edit Profile"
  description="Update your profile information"
  size="lg"
>
  <MobileModal.Content>
    {/* Form or content */}
  </MobileModal.Content>
  <MobileModal.Actions>
    <Button variant="outline">Cancel</Button>
    <Button>Save</Button>
  </MobileModal.Actions>
</MobileModal>
```
