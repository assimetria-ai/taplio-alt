## Usage Example

**Zero-Config CRUD:**
```javascript
const { createCrudRouter } = require('./lib/@system/Helpers')

// One line creates 5 endpoints
module.exports = createCrudRouter({
  repo: ProductRepo,
  config: { basePath: '/api/products' }
})

// Creates:
// GET    /api/products       - List with pagination
// GET    /api/products/:id   - Get single
// POST   /api/products       - Create
// PATCH  /api/products/:id   - Update
// DELETE /api/products/:id   - Delete
```

**Manual Control:**
```javascript
const { pagination, handleList } = require('./lib/@system/Helpers')

router.get('/api/products', pagination(), (req, res, next) => {
  handleList({ repo: ProductRepo, req, res, next })
})
```
